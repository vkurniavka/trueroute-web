# Street or Place Missing from Map

TrueRoute map tiles, place search, and points of interest are all built from **OpenStreetMap (OSM)** — a community-maintained open-source map. If something is missing or shows an outdated name, the data either has not been added to OpenStreetMap yet or needs to be updated there.

## Common symptoms

### Street is missing from the map entirely

The road exists in real life but is not drawn on the map.

**Cause:** The street has not been mapped in OpenStreetMap for this area yet.

**Fix:** Add the street to OpenStreetMap. See [How to fix missing data](#how-to-fix-missing-data) below.

### Street is on the map but not found in search

The street is visible when you look at the map but does not appear when you search for it by name.

**Cause:** The street exists in OpenStreetMap but is not yet indexed in the TrueRoute place search database, or the street name in OpenStreetMap is slightly different from what you searched.

**Fix:** Check [openstreetmap.org](https://www.openstreetmap.org) to confirm the street name is correct, then update it if needed.

### Place name is outdated or wrong

A street, neighbourhood, or locality is shown with an old name that was changed years ago.

**Cause:** OpenStreetMap still has the old name. Renaming streets in OpenStreetMap requires a contributor to update the record.

**Fix:** Update the name in OpenStreetMap. See [How to fix missing data](#how-to-fix-missing-data) below.

### Fuel station, pharmacy, hospital, or parking is missing

A point of interest exists in real life but does not appear in TrueRoute.

**Cause:** The location has not been added to OpenStreetMap, or it opened recently and has not been mapped yet.

**Fix:** Add the location to OpenStreetMap.

## Where TrueRoute map data comes from

| What you see | Source |
|---|---|
| Map tiles (streets, buildings, water) | OpenStreetMap via Geofabrik daily export |
| Place and street search | OpenStreetMap via Geofabrik daily export |
| Fuel stations, pharmacies, hospitals, parking | OpenStreetMap amenity data |

TrueRoute rebuilds its maps regularly from the latest OpenStreetMap data. Once a correction is made in OpenStreetMap, it will appear in TrueRoute when the next map update is published.

**OpenStreetMap data for Ukraine:** [download.geofabrik.de/europe/ukraine.html](https://download.geofabrik.de/europe/ukraine.html)

## How to fix missing data

Anyone can edit OpenStreetMap for free. Edits appear in TrueRoute after the next map update.

### Option A — StreetComplete (recommended for mobile)

StreetComplete is a mobile app that makes contributing to OpenStreetMap straightforward. It shows tasks near you — adding missing roads, correcting names, and filling in place details.

1. Install **StreetComplete** from Google Play.

2. Open the app and navigate to the area with missing or incorrect data.

3. Complete the task shown — for example, confirm a road name or add a missing street.

4. Submit the change. It will be reviewed and added to OpenStreetMap.

**Download:** [streetcomplete.app](https://streetcomplete.app)

### Option B — OpenStreetMap iD editor (desktop)

The iD editor is the browser-based editor on openstreetmap.org. Use it for more complex edits such as adding a new road, renaming a street, or adding a business.

1. Go to [openstreetmap.org](https://www.openstreetmap.org) and create a free account.

2. Navigate to the area you want to edit.

3. Tap **Edit** to open the iD editor.

4. Add the missing street, correct the name, or add the missing place.

5. Save your changes with a brief description.

## After your edit

OpenStreetMap changes are reviewed by the community and accepted within minutes to a few hours. TrueRoute rebuilds its maps regularly from the latest data. Your correction will appear in TrueRoute after the next map update — you do not need to reinstall the app or re-download the map.
